Getting started with the Academic theme for Hugo

Sun, Sep 4, 2016 academic, hugo

I use the Hugo static site generator with the Academic theme to create my personal website. Hugo is a static site generator, which means unlike dynamic site generators which build a page every time a visitor requests one, it builds all pages when you create your website. The academic theme is generally designed for academic staff and students. You can write in Markdown for easy formatting and code highlighting, with LaTeX for mathematical expressions.
